## 关于项目

它是一个非常棒的创意，是一款与 [Mikutap](https://aidn.jp/mikutap) 类似的板垫音乐游戏。为了降低国内用户访问的成本和自制并托管网站的难度，本项目应运而生。

## 修改说明
